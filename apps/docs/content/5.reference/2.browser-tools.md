# Browser Tools

BabyClaw can optionally control a browser through MCP (Model Context Protocol). This lets the agent navigate web pages, click elements, fill forms, and take screenshots. Browser tools are **disabled by default**.

## Enabling browser tools

Set `tools.enableBrowserTools` to `true` in your config:

```json
{
  "tools": {
    "enableBrowserTools": true,
    "browser": {
      "headless": true
    }
  }
}
```

Browser tools require an MCP browser server to be available. BabyClaw connects to it and exposes the browser actions as agent tools.

## Available tools

| Tool | What it does |
|---|---|
| `browser_navigate` | Open a URL in the browser |
| `browser_get_state` | Get the current page state (URL, title, content) |
| `browser_click` | Click an element on the page |
| `browser_type` | Type text into an input field |
| `browser_screenshot` | Take a screenshot of the current page |

## Headless vs visible

By default, the browser runs in headless mode (no visible window). If you want to see what the browser is doing:

```json
{
  "tools": {
    "browser": {
      "headless": false
    }
  }
}
```

## Use cases

Browser tools are useful when the agent needs to:

- Fill out web forms
- Navigate authenticated web apps
- Scrape pages that require JavaScript rendering
- Take screenshots for visual confirmation
- Interact with web-based tools that don't have APIs

## Model selection

Browser automation tasks can be token-intensive (page content, screenshots). The `ai.models.browser` model is used for browser-related reasoning. You can set this to a different model than your main chat model if you want:

```json
{
  "ai": {
    "models": {
      "chat": "anthropic:claude-sonnet-4-20250514",
      "browser": "anthropic:claude-sonnet-4-20250514"
    }
  }
}
```

## Limitations

- Browser tools need an MCP server running alongside the gateway
- Only one browser instance is managed at a time
- Some sites with aggressive bot detection may not work well
- Screenshots are sent as images, which requires a model with vision support
